POP3 Conversation
This is a simple conversation between a server and a client for POP3:



POP protocol (+ = response)

USER you@ISP.com

+OK Password required

PASS your_password

+OK USER ********** logged in at (Time)

STAT

+OK 1 xxxx
LIST
+OK (? number) messages

RETR (# of message)

+OK



.

DELE (# of message)

Quit

End Of Conversation
